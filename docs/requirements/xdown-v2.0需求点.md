# v2.0 阶段：性能核心与多协议支持

_目标：解决“下得慢”的问题，并支持主流的极客下载协议。_

- **3.2.1 多线程分块下载 (HTTP/HTTPS)**
  
    - 利用 HTTP `Range` 请求头，将单个大文件动态切分为多个 Chunk，开启多个线程同时下载，最后合并。
    
- **3.2.2 P2P 下载协议支持**
  
    - 引入第三方核心库 `libtorrent`，解析 `.torrent` 种子文件和 `magnet:` 磁力链接。
      
    - 获取 Tracker 和 DHT 节点信息，实现基础的 BT 互传。
    
- **3.2.3 磁盘与文件管理优化**
  
    - **磁盘预分配**：下载启动时，根据文件大小预先向操作系统申请全量磁盘空间，防止磁盘碎片化。
      
    - **完整性校验**：下载完成后，根据服务端提供的 ETag 或 Hash 值进行文件完整性校验。
